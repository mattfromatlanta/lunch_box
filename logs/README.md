# Logs Directory

This directory contains log files generated by the Lunch Box application.

## Log File Format

Log files are automatically created with timestamps:
```
lunch_box_log_YYYYMMDD_HHMMSS.txt
```

Example: `lunch_box_log_20260131_175751.txt`

## Contents

Each log file contains:
- Application initialization messages
- Target folder path
- Scan results (number of files found)
- File details (name, size, audio properties)
- Error messages (if any)

## Notes

- Log files are created automatically on each program run
- Files in this directory are ignored by git (except this README and .gitkeep)
- Logs are useful for reviewing past scans and debugging
