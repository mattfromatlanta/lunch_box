# Test Audio Files

This directory contains WAV files for testing the Chompi Pack audio scanner.

## Directory Structure

- **mono/** - Mono (single channel) WAV files
- **stereo/** - Stereo (2 channel) WAV files
- **16bit_44k/** - 16-bit 44.1kHz WAV files (CD quality)
- **24bit_48k/** - 24-bit 48kHz WAV files (professional audio)
- **various/** - Mixed formats for general testing
- **nested/subfolder/** - Test recursive directory scanning

## Notes

- Place your test WAV files in the appropriate subdirectories
- The scanner will recursively search all subdirectories
- Use various bit depths (16-bit, 24-bit, 32-bit) and sample rates (44.1kHz, 48kHz, 96kHz) for comprehensive testing
