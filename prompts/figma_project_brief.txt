SECTION 1: PROJECT OVERVIEW
────────────────────────────────────────────────────────────────────────────────

Chompi Pack is a desktop audio sample processor for the CHOMPI sampler hardware.
It converts audio files to CHOMPI-compatible format (16-bit, 48kHz WAV) and
organizes them using the CHOMPI bank/slot naming convention. The app runs on
macOS as a native desktop application built with the JUCE C++ framework.

TARGET USERS
  Audio producers and musicians who own the CHOMPI sampler hardware. They prepare
  sample libraries on their computer, then transfer the output files to the device.
  Users range from casual to technical, skewing toward musicians who are
  comfortable with file management but not necessarily developers.

CORE WORKFLOW
  1. User selects or drags audio files/folders into the app.
  2. Files are assigned to banks (A–E) and slots (1–14) within one of two
     categories: Cubbi (percussive) or Jammi (chromatic).
  3. User sets an output folder path.
  4. User clicks "Process Samples" — the app converts and exports files in the
     CHOMPI naming format (e.g. cubbi_a1.wav, jammi_b3_double.wav).
  5. User opens the output folder and transfers files to the CHOMPI hardware.

CHOMPI HARDWARE CONSTRAINTS (hard limits that drive every layout decision)
  - 2 categories: Cubbi, Jammi
  - 5 banks per category: A, B, C, D, E
  - 14 slots per bank
  - 70 samples max per category (5 × 14)
  - Output: 16-bit, 48kHz WAV only
  - Each sample produces a base file AND a _double (octave-up) file
  - Accepted input formats: WAV, AIFF, MP3, FLAC

  Arrow keys:       Move focus cell (single selection)
  Shift + Arrow:    Expand rectangular selection
  Tab:              Move focus down one bank (same slot column)
  Space:            Play/stop preview of focused cell
  Delete/Backspace: Clear selected cells
  Enter/Return:     Browse for file at focused cell
  Cmd + click:      Toggle-add cell to selection
  Shift + click:    Range-select to clicked cell

DRAG BEHAVIORS (Advanced mode)
  - Drag a filled slot onto empty slot: moves the file
  - Drag a filled slot onto filled slot: swaps the two files
  - Hold Cmd while dragging: swap mode (displaced files move back to vacated cells)
  - While dragging, destination cells preview the post-drop state
  - Drag-select: click-drag across cells to select a rectangular range
  - Selection drag: click-drag on an already-selected cell to move the selection

BEHAVIORS
  - Switching category tab stops any active preview playback
  - Grid state persists when switching between Cubbi and Jammi tabs
  - Switching from Advanced to Bank mode syncs grid data to Bank view
  - Right-click slot: context menu (Browse, Preview, Clear)
  - Drop audio file from OS onto slot: assigns it directly
